News: Astriarch - Ruler of the Stars



Latest Announcements on MasteredSoftware.com

Apr 21, 2014: New releases will be announced going forward on the MasteredSofware.com blog.
View all the details of the latest releases to see what has been recently changed.


HTML5 Version Open-Sourced

Feb 2, 2014: The current HTML5 version of Astriarch - Ruler of the Stars has just been open-sourced!

Aspiring Game Developers can use the source code as a reference for creating their own HTML5 canvas games.
You can download the source code from GitHub here.


Version 1.2.1 Released

Jul 20, 2013: Astriarch - Ruler of the Stars v1.2.1 released.

This version fixes browser incompatibilities with a javascript library providing stylized radiobuttons and checkboxes.


Astriarch Silverlight Version Open-Sourced!

Nov 13, 2012: The original Silverlight version of Astriarch - Ruler of the Stars has just been open-sourced!

The open source code corresponds to the final Silverlight version 1.0.1 release, it is intended to provide an example of how to create a turn-based strategy game in Silverlight.
You can download the source code from GitHub here.


Version 1.2.0 Released

May 18, 2012: Astriarch - Ruler of the Stars v1.2.0 released.

In version 1.2.0 mineral resources (Iridium and Ore) are now stored at the planet level, when planets change owners, Iridium, Ore, Food and Gold may be looted by the new owner. This version also fixes some minor bugs and graphical problems.


Version 1.1.0 Released

Feb 1, 2012: Astriarch - Ruler of the Stars 1.1.0 has just been released.

Version 1.1.0 automatically saves the current game at the end of each turn and allows the saved game to be resumed. There are also a number of minor bug fixes and other enhancements.


Astriarch Web (HTML5/JavaScript) beta version released

Jul 7, 2011: Astriarch has just been ported from Silverlight to pure HTML5/JavaScript and the beta version has been released on the Chrome Web Store. This fully rewritten version works in Internet Explorer 9, FireFox, Chrome, and Safari.


Version 1.0.1 Released

May 26, 2011: Astriarch - Ruler of the Stars version 1.0.1 has just been released.

Version 1.0.1 includes new features to improve playability. There are now planet strength indicators shown above each planet as well as a new feature to keep building the last ship built on each planet. The easy computer AI is now slightly less aggressive and a bug related to the AI build logic has been fixed.


Astriarch Gold Release (v1.0.0)

May 3, 2011: Astriarch - Ruler of the Stars is now Gold! A stellar ambient soundtrack has been added and other game balance improvements have been made. Special thanks to Resonant for composing the excellent music. Check back soon for tablet versions of Astriarch - Ruler of the Stars.


Astriarch 0.9.7 Released

Oct 19, 2010: Version 0.9.7 has been released, the AI has been improved and a few minor bugs have been fixed.


Astriarch 0.9.6

Oct 7, 2010: Version 0.9.6 has just been released, this version fixes a few minor bugs and adds new buttons on the send ships window to easily send all ships on the planet.


Starship advantages - 0.9.5

Oct 3, 2010: Version 0.9.5 has just been released, now each ship gets an strength bonus when attacking ship types it has an advantage against.

Astriarch Tutorial Videos now Posted

Sep 28, 2010: The tutorial videos have now been published, these should give new players a good introduction to some of the mechanics of the game as well as some hints on how to beat the computer. Enjoy!


Introducing Astriarch - Ruler of the Stars

Sep 24, 2010: Astriarch - Ruler of the Stars is the new name for the space strategy game System Master. We've just finished the move from the old location to our new domain.
